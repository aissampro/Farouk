## Qu'est-ce qu'AI CEO OS ?

**AI CEO OS** n'est pas un chatbot. C'est un **constructeur de startup IA** qui simule une équipe fondatrice complète.

Tu fournis une idée de business. Le système agit simultanément comme :
- 🎯 Un **consultant stratégique niveau McKinsey** — analyse de marché, positionnement, modèle économique
- 🏢 Un **CEO de startup** — feuille de route, structure d'équipe, prise de décision
- 🤖 Une **équipe de 4 agents IA** — Marketing, Ingénierie, Ventes, Ops — travaillant en parallèle
- 📦 Un **générateur de livrables** — plan d'affaires, stratégie GTM, roadmap produit, plan d'action

Le tout, entièrement bilingue **anglais / français**.

## ✨ Fonctionnalités

| Fonctionnalité | Description |
|---|---|
| 🔍 Analyse Stratégique | Opportunité de marché, utilisateurs cibles, positionnement, modèle économique, risques |
| 🗺️ Plan CEO | Roadmap 90 jours, structure org, assignations agents, KPIs |
| 🤖 4 Agents IA | Marketing, Ingénierie, Ventes, Ops — en parallèle |
| ✅ Orchestration | Tâches auto-générées, assignées aux agents, traçables |
| 📄 Rapport Consolidé | Stratégie GTM, roadmap produit, projections revenus, actions immédiates |
| 🌍 Bilingue | Support complet EN / FR — UI, prompts et outputs IA |
| ⚡ Streaming | Streaming token par token en temps réel |
| 🎛️ Fichier Unique | Zéro dépendances — fonctionne directement dans le navigateur |

## 🚀 Démarrage Rapide

### Option 1 — Lancer instantanément (sans installation)

Ouvre `src/index.html` directement dans ton navigateur. C'est tout.

> Tu as besoin d'une [clé API Anthropic](https://console.anthropic.com/).

### Option 2 — Cloner et lancer en local

```bash
git clone https://github.com/YOUR_USERNAME/ai-ceo-os.git
cd ai-ceo-os
open src/index.html
```

### Option 3 — Déployer sur Vercel / Netlify

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/ai-ceo-os)

## 🤖 L'Équipe d'Agents IA

| Agent | Rôle | Responsabilités |
|---|---|---|
| **Agent Marketing** | Croissance & Marque | Proposition de valeur, stratégie contenu, branding, copy landing page |
| **Agent Ingénierie** | Produit & Tech | Architecture MVP, priorisation features, stack technique |
| **Agent Commercial** | Revenus & Prospection | Définition ICP, prospection, tactiques de conversion |
| **Agent Ops** | Exécution & Process | Workflows, métriques, parcours client, automatisation |

---

## 🌍 Support Bilingue

Bascule entre **anglais** et **français** via le toggle `EN / FR` dans l'en-tête.

En mode français :
- L'interface bascule entièrement en français
- Tous les prompts Claude sont envoyés en français
- Tous les outputs IA sont générés en français
- Les noms d'agents s'adaptent : "Sales Agent" → "Agent Commercial"
