## [1.0.0] - 2026-05-02

### Added
- Initial release of AI CEO OS
- 5-phase orchestration: Strategy → CEO Plan → Tasks → Agents → Outputs
- 4 AI agents running in parallel: Marketing, Engineering, Sales, Ops
- Real-time token streaming for all AI-generated content
- Full bilingual support: English and French (UI + prompts + outputs)
- Live language switcher (EN / FR) in the header
- Quick-start example ideas (bilingual)
- Task orchestration with auto-assignment and progress tracking
- Activity log with timestamps
- Agent status tracking (idle / working / done)
- Consolidated output report with GTM, roadmap, revenue projections
- Zero-dependency single-file implementation
